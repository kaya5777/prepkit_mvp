## How to run locally
1. Install dependencies
```bash
bundle install
```

2. Set your OpenAI API key
```bash
export OPENAI_API_KEY=sk-...  # replace with your OpenAI API key
```

3. Start the server and open the app
```bash
rails server
```
Then open `http://localhost:3000` in your browser.
